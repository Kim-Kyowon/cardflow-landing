Features

필요한 기능만,
군더더기 없이

복잡한 가계부 앱 대신, 카드 실적 관리에 집중한 심플한 앱입니다.

🔔

자동 푸시 알림 파싱

카드사 앱 푸시 알림을 수신하는 즉시 자동으로 카드사·금액·가맹점을 인식합니다. 별도 입력 없이 자동 등록됩니다.

📊

카드별 실적 진행률

카드별 당월 목표 금액 대비 달성률을 프로그레스 바로 한눈에 확인하세요. 어느 카드를 더 써야 할지 바로 보입니다.

🗂

카테고리 자동 분류

식비·교통·쇼핑·의료·구독·공과금 등 7개 카테고리로 자동 분류합니다. 실적 제외 카테고리도 카드별로 설정 가능합니다.

📅

월별 내역 조회

월·카드·카테고리별 필터로 원하는 내역만 빠르게 조회하세요. 일별 그룹핑으로 소비 흐름을 파악할 수 있습니다.

✏️

수기 내역 등록

현금 결제나 알림을 놓친 내역도 직접 등록할 수 있습니다. 카테고리와 날짜를 자유롭게 지정하세요.

🔒

완전 로컬 저장

모든 데이터는 내 폰에만 저장됩니다. 서버 전송·회원가입 없이 사용하세요. 인터넷 연결도 필요 없습니다.

How it works

사용법, 딱 3단계

복잡한 설정 없이 설치하고 권한만 허용하면 모든 게 자동입니다.

1

APK 설치

아래 APK 파일을 다운로드해 설치합니다. 설치 가이드를 참고하면 5분이면 완료됩니다.

2

알림 접근 권한 허용

앱 첫 실행 시 알림 접근 권한을 허용합니다. 카드사 푸시를 읽는 핵심 권한입니다.

3

카드 결제 후 자동 등록

카드를 사용하면 자동으로 내역이 등록됩니다. 앱을 열어 실적을 확인하세요.

Install Guide

설치 가이드

플레이스토어 미등록 APK이므로 아래 단계를 따라 설치해주세요.

STEP 1

APK 파일 다운로드 후 설치

상단 APK 다운로드 버튼을 눌러 구글 드라이브 페이지에서 파일을 받고, 파일 관리자에서 실행하세요.
크롬 브라우저의 경우 다운로드 알림에서 바로 설치할 수 있습니다.

💡 설치 전 설정 → 보안 → 출처를 알 수 없는 앱에서 브라우저(또는 파일 관리자)의 설치 허용을 켜주세요.
STEP 2

"안전하지 않은 앱" 경고 무시하기

플레이스토어를 거치지 않은 APK이기 때문에 Android가 경고를 표시합니다.
"무시하고 설치"를 눌러 설치를 진행해주세요.

🔒 Cardflow는 모든 데이터를 기기 내부에만 저장합니다. 서버로 전송되는 정보는 없습니다.
STEP 3

Play 프로텍트 검사 끄기 (선택)

Play 프로텍트가 앱을 유해 앱으로 경고하는 경우 Play 스토어 → 메뉴 → Play 프로텍트 → 설정에서 "Play 프로텍트로 앱 검사"를 일시적으로 꺼주세요.

💡 정식 마켓 출시 후에는 다시 켜두시는 것을 권장합니다.
STEP 4

배터리 최적화 중지 · 알림 허용

앱 설치 후 첫 실행 시 두 가지 팝업이 표시됩니다.

  • 배터리 사용량 최적화 중지 → 허용 선택
    (백그라운드에서 알림을 안정적으로 수신하기 위해 필요합니다)
  • 알림 허용 → 허용 선택
STEP 5

알림 접근 권한 허용

앱 화면에서 "알림 접근 권한 허용하기" 버튼을 누르면 시스템 설정 화면으로 이동합니다.
목록에서 Cardflow를 찾아 토글을 켜고 "허용"을 눌러주세요.

✅ 허용 후 앱으로 돌아오면 자동으로 메인 화면이 시작됩니다. 이제 카드를 사용하면 자동으로 내역이 등록됩니다!
